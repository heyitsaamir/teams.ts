# Release Process

1. Go to the [Releases page](https://github.com/microsoft/teams.ts/releases) and create a new release with a version tag (e.g., `v2.0.6`)

2. Trigger the release pipeline: [Azure DevOps Pipeline](https://dev.azure.com/DomoreexpGithub/Github_Pipelines/_build?definitionId=46&_a=summary)
